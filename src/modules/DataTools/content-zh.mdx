# 数据处理工具对比

## 概述

在数据分析项目中，选择合适的工具至关重要。本模块将深入对比 Python 中最常用的三种数据处理工具：List、NumPy 和 Pandas，帮助您根据具体需求选择最佳工具。

## 工具性能对比示例

通过实际代码示例了解三种工具的性能差异：

```python
import time
import numpy as np
import pandas as pd

# 创建测试数据
size = 1000000
data = list(range(size))

# 1. List 处理
start_time = time.time()
list_result = [x * 2 for x in data if x % 2 == 0]
list_time = time.time() - start_time

# 2. NumPy 处理
np_data = np.array(data)
start_time = time.time()
np_result = np_data[np_data % 2 == 0] * 2
np_time = time.time() - start_time

# 3. Pandas 处理
df = pd.DataFrame({'value': data})
start_time = time.time()
pandas_result = df[df['value'] % 2 == 0]['value'] * 2
pandas_time = time.time() - start_time

print(f"List 处理时间: {list_time:.4f}秒")
print(f"NumPy 处理时间: {np_time:.4f}秒")
print(f"Pandas 处理时间: {pandas_time:.4f}秒")
print(f"\n结果数量: {len(list_result)}")
```

## CSV 数据处理示例

```python
# CSV 数据处理示例
import pandas as pd
import numpy as np

# 读取 CSV 文件
df = pd.read_csv('sales_data.csv')

# 数据清洗
df = df.dropna()  # 删除缺失值
df = df[df['销量'] > 0]  # 过滤异常数据

# 数据分析
total_sales = df['销售额'].sum()
avg_price = df['单价'].mean()
top_product = df.groupby('产品')['销量'].sum().idxmax()

print(f"总销售额: {total_sales:,}")
print(f"平均单价: {avg_price:.2f}")
print(f"最畅销产品: {top_product}")
```
